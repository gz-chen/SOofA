DESCRIPTION
	The file "program_2.tar.gz" contains computer programmes
	for producing Figure 1, Figure 2, Table S8, Table S9 and Table S10
	in the article "Stratum order-of-addition designs"
	by Liushan Zhou, Ze Liu, Min-Qian Liu and Guanzhou Chen.
	It also contains a programme for producing some of the known stratum OofA designs.
	For a detailed description, please see the text file "README.txt" in "program_2.tar.gz".
